Training

Find out about and access training to suit your govCMS needs.

Online and face to face tutorials, information sessions and documentation are available, cover all topics and address specific roles and functions. Below are some key sources of online training you can use as a govCMS customer.

e-learning videos

Introductory and basic training on the features and functions of govCMS is available in the e-learning videos here and help you to know what govCMS is, how it works, and how you can use it.

govCMS solutions and features overview

An overview of govCMS and a demonstration of the features and functions of a standard installation, covering standard tasks and procedures involved to create, edit, approve and publish content

Create and modify content

How to create and modify a standard page as a Content Editor using a standard template

Create and configure site settings

How to use the configuration part of govCMS as a Site builder and/or editor to create and manage settings. It shows you how to use each of the settings features of the configuration function. These features are titled; People, Content authoring, Media, Search and metadata, System, User interface, and Development.

Maintain up to date site and page links and URLs

How to manage and maintain consistent and up to date links and urls for your sites. It shows site builders and editors how to: manually create a URL redirect, correct bad links, and update existing page URLs

Administer user accounts

How to use the features of the People component of the system to manage all aspects of user account administration.

Create and use webforms

How to use the functions of the webforms component to create contact forms, surveys and customer feedback pages for your sites. This video covers how to: develop checkboxes and fields using the Form Components function, set rules using the Conditional function, and generate access/limits/control values using the Settings function

Schedule content for release or removal

How to use the features in the Scheduling component to publish or un-publish specific content at a set day or time in the future
